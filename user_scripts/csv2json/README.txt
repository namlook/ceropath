CSV2JSON
========

Just copy the csv2json directory, put all the csv into the csv directory and
launch the csv2json.py script. Wait for it and your done ! The json.zip can be
uploaded to the website.


Requirements
------------

csv2json.py require python and the following python modules:

 * pyyaml
 * anyjson

Just install easy_install to install them (ahem!)....
